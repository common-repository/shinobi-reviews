A review plugin for gathering many customer reviews easily.

== Description ==

Shinobi Reviews can gather and manage many kinds of reviews. For example, if you want to add customer reviews in your site, all you need to do first is just create a form and display it in your site. After that, you need to do is just wait to be posted reviews.

= Demo =

If you want to try this plugin before activation, you can use it on [English version](https://shinobireviews.com/) or [Japanese version](https://shinobireviews.com/ja/)

= Anti-Spam =

Don't worry if you are concerned about posting spam reviews. You can use Google reCAPTCHA and it blocks spam reviews.

= Membership System =

There is a membership system in this plugin. You can use it to anti-spam not only reviewer management.

= Administrator can manage =

The posted review can be managed by the administrator. Unnecessary reviews can be deleted or modified.

== Frequently Asked Questions ==

= How do I use it? =

1. Create a review form in admin
2. Copy the shortcode displayed after saving
3. Paste the copied shortcode into the article
4. Users can post reviews on your website

= Which browser works? =

This plugin works with modern browsers and Internet Explorer 11.
